Nodemono

Kimono, ehhh, I mean NODENOMO is a webscraping tool that allows you to extract data from almost any webpage.

It's easy to use and doesn't require any intense programming skills, plus it's open source for you. You just need a browser with chrome-extension and then you can get scraping! Kimono users have created incredible apps and sites ranging from price comparisons to sports analytics. If you need some inspiration, take a look at our gallery.

To get started, install the chrome extension, navigate to the site you'd like to scrape, and start clicking!
