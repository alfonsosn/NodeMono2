Modify Results

You can write a javascript function to transform your results to your liking. The function you write takes in the data argument, a javascript object representation of your APIs JSON endpoint. The modified version of your JSON endpoint will consist of whatever your functions returns. You can access this modified version of your data in the 'Use Data' tab. When retrieving modified results, the only available output format is JSON.
